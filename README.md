## showreel-editor

ショーリール編集のUIプロトタイプです。クリップの一覧からタイムラインに追加し、
合計再生時間の表示やフォーマット整合性の確認など、編集フローの基本要素を検証できます。

### 環境

- Bun

### セットアップ

```bash
bun install
```

### 開発サーバーの起動

```bash
bun run dev
```

### ビルド / プレビュー

```bash
bun run build
bun run preview
```

### テスト

ユニットテスト:

```bash
bun run test
```

E2E（Playwright）:

```bash
bun run test:e2e
```

E2E UI:

```bash
bun run test:e2e:ui
```
